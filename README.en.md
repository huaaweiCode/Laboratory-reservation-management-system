# 实验室预约管理系统

#### Description
实验室预约管理系统拥有三个角色

- 管理员：查看实验室、公告管理、教师管理、学生管理、实验室排期表、新增实验室、预约管理等
- 教师：根据条件查询个人预约记录、添加管理课程、预约（个人或者课程）实验室
- 学生：查看预约信息、个人预约实验室、实验室预约详情等
